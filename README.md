# Flashcard Learning App

A simple GUI-based flashcard learning application built using Python and Tkinter.

## Features
- Randomized flashcards
- Flip to reveal answer
- Next card navigation
- Clean GUI layout

## Technologies Used
- Python
- Tkinter
- Random module

## How to Run

Make sure Python is installed.

Run:
```bash
python main.py

```
## Purpose
Built as part of academic coursework to demonstrate GUI development and Python fundamentals.
